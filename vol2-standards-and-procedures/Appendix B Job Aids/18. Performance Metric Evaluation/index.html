    Observed Data¶

    An annual maximum series of the subbasin average precipitation for each duration are calculated for the project. This data is computed during the scoping and data preparation phase of the FFRD process and used for quality assurance of the observed data. [as described in section X. data is stored in X]

    Modeled Data¶

    Average precipitation for each event over multiple durations for each subbasin for each event will be calculated. [reference tool available for mining this data. Bartles java script?]
